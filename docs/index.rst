Introduction
------------
shapleyx is a Python package for global sensitivity analysis using Sparse Random Sampling - High Dimensional Model Representation (HDMR) with Group Method of Data Handling (GMDH) for parameter selection and linear regression for parameter refinement.

Key Features:

- Global sensitivity analysis using Sparse Random Sampling HDMR
- Support for Sobol indices, Shapley effects, and PAWN indices
- Legendre polynomial expansion for basis functions
- Multiple regression methods (ARD, OMP)
- Bootstrap resampling for confidence intervals
- Prediction capabilities using surrogate models
